- [ft_malcolm](https://github.com/mrlouf/ft_malcolm) - ARP spoofing in C
- [dr_quine](https://github.com/mrlouf/dr_quine) - Self-replicating programs
- [ft_ls](https://github.com/mrlouf/ft_ls) - Recreation of the ls command

- [42sh](https://github.com/mrlouf/42sh) - Shell implementation
- [darkly](https://github.com/mrlouf/darkly) - Web security CTF project

## Cloning This Repository
To clone this repository with all submodules:
```bash
git clone --recurse-submodules https://github.com/mrlouf/42-outer-core.git
```

Or, if you've already cloned the repository:
```bash
git submodule update --init --recursive
```
